Female Uniform Detection System

Introduction


The issue is that departmental disciplinary officers have to inspect each student's uniform to ensure discipline, which is a time-consuming operation to ensure that every student in the department or institution is dressed appropriately.Our uniform detection system allow us to distingusih uniform students from non uniform students as soon as they came across camera's eye.

System Goals

  • Maintains the transparency in the detection process of proper dress code
  • Cope with the issue of time consumption
  • Facilitate disciplinary officers

    • System Objectives

  • Maintenance of uniform regulation
  • User friendly interface
  • Reduce cognitive workload on user end

    • .01 Operating Environment

    This system will operate in the "Windows Operating System” including its software components like “Anaconda” and “visual studio code”. This detection process is done on any "Laptop" with Intel Corei5 processor and its latest versions and a camera for image detection.

    .02 Hardware Interface

    • Supported device types: Laptop (Processor: Intel(R) Core (TM) i7-7500U CPU @ 2.70GHz 2.90GHZ, Total Memory: 20 GB (Free Space), RAM: 8GB, OS: Windows 10 PRO 64-bit). Camera (32 Mega Pixels, Resolution: 1280 x 720 pixels). Nature of Data: Images.

    .03 Software Interface

    Software Components: Python (3.7), OS: Windows (Windows10), Tools: CVAT, Libraries: TensorFlow (2.0), Pandas (1.0), OpenCV (3.0), Keras (2.4.0) Integrated commercial components: SSD

    Our Work

    Scope

    •This system will detect female uniform.

    • This system will only work for the female students of UOS.

    • This system will only work in controlled environment (i.e. proper light).

    • This system is designed to detect dress code for arts and science groups.

    •This system will only label “Proper Uniform” for those students whose uniform is according to the dress code mentioned in the university’s prospectus.

    • This system will be designed for individual students.

    • This system will not able to detect the uniform of male students.

    This system will not detect blurred videos.

    Functional Requirement's

    System must be able to capture video properly.

    System must be able to extract image from video frames.

    System must be able to extract ROI from frame.

    • System must be able to load data set.

    • System must be able to classify images on base of classifiers.

    • System must be able to Label students and show it on screen from where it can be monitor.
